搭建自己的博客

标签（空格分隔）： 博客 阿里云 github Hexo

目标:搭建一个自己的博客,经过一系列挑选,最终选择Hexo.（优点小巧,方便,支持github）我有台阿里云服务器.介于公司没代理没法上网,而且上网非常慢.就采用博客放在阿里云上并且可以和github上的gitpage同步.当然如果在本地搭建的话也可以直接配置本地同步到github上.这样能在github上直接访问配置后的页面.

阿里云配置

CPU： 1核
内存： 1024 MB
实例类型： I/O优化
操作系统： CentOS 7.3 64位

需求

  1. 在阿里云服务器安装Hexo
  2. 同步Hexo到GitPages

第一部分—安装Hexo
首先一般安装软件的过程是首先下载软件包,安装具体依赖包,然后编译安装具体软件.Hexo安装依赖nodejs所以首先要安装nodejs
登陆nodejs网站：https://nodejs.org/en/
首先看到首页下载链接




点击Download

我们下载大多数人使用的V6.10.3版本

下载源代码

mkdir package
cd package/
wget https://nodejs.org/dist/v6.10.3/node-v6.10.3.tar.gz
tar -zxvf node-v6.10.3.tar.gz

安装nodejs
那接下来我们要安装了,一般安装一个东西前需要安装该软件的依赖包,nodejs依赖gcc gcc-c++ openssl-devel

安装好可以开始编译了


设置环境变量

让变量修改生效.完成,验证是否生效

准备活动做完接下来可以开始安装Hexo了
hexo安装是通过nodejs下的npm包来安装的,默认安装完nodejs后就会有npm了
npm默认不是国内源,切换到淘宝源

执行 应该用cnpm

安装好后开始创建博客文件夹

最后可以看到博客安装成功了

这里的地址是我阿里云的地址.
